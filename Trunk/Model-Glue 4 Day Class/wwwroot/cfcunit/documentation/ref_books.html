This is a listing of useful printed resources on testing.

  • Beck, Kent. Test-Driven Development: by example.
    Addison-Wesley: Boston, 2003.
  • Link, Johannes. Unit Testing In Java: how tests drive the code.
    Morgan Kaufmann: San Francisco, 2002.
  • Hamill, Paul. Unit Test Frameworks.
    O'Reilly Media, Inc.: Sebastopol, CA, 2005.
  • Massol, Vincent. JUnit in Action.
    Manning Publications Co.: Greenwich, CT, 2004.
  • Rainsberger, J. B.. JUnit Recipes: Practical Methods for Programmer Testing.
    Manning Publications Co.: Greenwich, CT, 2005.
