cfcUnit Testing Framework

cfcUnit is a new member of the xUnit family of unit testing frameworks, written for the CFMX environment using ColdFusion Components and based entirely on the API of the JUnit testing framework. Unit testing is an important part of the software development cycle and the availability of such tools for ColdFusion developers leaves much to be desired. cfcUnit was built to fulfill the needs of a basic regiment of unit testing while providing the flexibilty to be extended to whatever custom testing solution you can think of. For more about unit testing, you can read Christian Cantrell's description of unit testing and why you would use it.
